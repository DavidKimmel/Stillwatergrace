4 Data Flow

Daily automated pipeline — triggered at 6:00 AM EST

1
Discover
6:00 AM EST — Cron Trigger
  • Fetch daily Bible verse (random book rotation)
  • Scrape Reddit hot posts (6 subreddits)
  • Pull Google Trends (faith/family keywords)
  • Scrape Pinterest trending pins
  • Score all content by virality potential → trending_content table
2
Generate
After Discovery completes
  • Select top 3 trending topics
  • Match to content calendar slots
  • Generate content via Claude API (all caption variants + hashtags + image prompt)
  • Store to generated_content (status: pending)
3
Image
After Generation completes
  • Submit image prompt to Leonardo.ai
  • Poll until complete → download raw image
  • Send to Canva API for brand overlay → export multi-format
  • Upload finals to Cloudflare R2
  • Store URLs to generated_images table
4
Queue
Dashboard Approval
  • Content appears in admin dashboard for human review
  • Approved content → scheduled_at timestamp set
5
Post
6:30 AM, 12:00 PM, 7:30 PM EST
  • Instagram → Facebook → TikTok
  • Log to posting_log
  • Send email notification
6
Analyze
1 hr, 24 hr, 7 days after posting
  • Pull Instagram Insights → analytics_snapshots
  • Weekly competitor scrape
  • Feed performance data back to content scoring
